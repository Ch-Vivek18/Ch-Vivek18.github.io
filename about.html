About SportPulse

SportPulse is an academic capstone project created as part of a learning exercise in web development. It is not a real website but a demonstration of front-end design concepts using HTML, CSS, and JavaScript. The project is designed to showcase skills in building a responsive sports-themed website with features like live scores, news updates, and video highlights. SportPulse is solely for educational purposes and does not provide real-time sports data or services.
